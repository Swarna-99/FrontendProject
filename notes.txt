Local storage:
1.Local Storage is a web storage feature that allows websities to store data in the browser with no expiration time.
2.This means the data remains availabele/even after the user closes and reopens the browser or the system is restarted.
3.It is tpicall used to save long-term data such as user preference,themes, or login tokens.


Session Storage:

1.Session storage is also a browser - based method, but it only stores data for the duration of a single browser tab session.
2.Once the tab is closed, all sessions storage data is automatically deleted.
3.It is useful for storing temparary data like form inputs or navigation steps that dont need to persist beond the current session.
4.Unlike local storage, session storage is not shared across tabs - each tab has its own session.


Git:-

Git is a distributed version control system that helps ou to track changes in your code, save different versions of your project, and work with others
without overwriting each other's work. It allows ou to create brancehes, experiment safety and easily go back to older versions whenever needed.
